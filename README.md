# DEMON-BOT 🤖

A powerful and feature-rich Discord bot designed to enhance your server experience with advanced automation, moderation, and entertainment features.

## Features ✨

- **Automation:** Automate repetitive tasks and save time
- **Moderation Tools:** Advanced moderation commands for server management
- **User Engagement:** Interactive commands and fun activities
- **Real-Time Monitoring:** Track and monitor server activities
- **Customizable:** Easily configure settings for your needs
- **Performance:** Fast and efficient with minimal resource usage

## Installation 🔧

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- A Discord bot token

### Setup Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vhmdtvmr112/DEMON-BOT.git
   cd DEMON-BOT
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure the bot:**
   - Create a `.env` file in the root directory
   - Add your Discord bot token and other settings:
   ```
   DISCORD_TOKEN=your_token_here
   PREFIX=!   
   ```

4. **Run the bot:**
   ```bash
   npm start
   ```

## Usage 🚀

Once the bot is running, you can use commands by mentioning the bot or using the configured prefix in your Discord server.

### Common Commands
- `!help` - Display available commands
- `!ping` - Check bot latency
- `!info` - Get bot information
- `!setup` - Configure bot for your server
